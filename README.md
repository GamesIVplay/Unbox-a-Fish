# 🧠 Trap and Bait a Brainrot

A Roblox game where you:

- Buy traps from the conveyor belt 🪤
- Place them on your plot to catch random Brainrots
- Brainrots earn money passively 💰
- Other players can steal your Brainrots
- Buy bait to boost trap speed, size luck, or rarity

Rebirth to increase plot capacity, unlock tools, and earn more money.  
Main goal: collect rare Brainrots and dominate the leaderboard.
